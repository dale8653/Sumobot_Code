# Sumobot Code

This repository contains the source code used for our **Sumobot project**.

## Hardware Used
- ESP32 Microcontroller
- L298N Motor Driver
- DC Gear Motors
- IR Sensors
- Ultrasonic Sensor
- OLED Display (128x64)
- Li-ion Battery Pack

## How to Use
1. Open `sumobot.ino` in **Arduino IDE**
2. Select **ESP32 board**
3. Install required libraries:
   - Adafruit GFX
   - Adafruit SSD1306
4. Upload the code to the ESP32

## Project Purpose
This code enables autonomous detection, navigation, and pushing behavior for Sumobot competition.
